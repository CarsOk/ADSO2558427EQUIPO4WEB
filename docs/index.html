Documentación

Software para la administración y operación de Domitransp SAS a través de aplicativo web y móvil

Introducción

Domitransp SAS es una empresa líder en logística y transporte. Este proyecto tiene como objetivo la creación de un software de gestión para mejorar la eficiencia operativa mediante soluciones web y móviles.

Justificación

Ante los desafíos operativos de Domitransp SAS, desarrollaremos un aplicativo web (Ruby on Rails) y móvil (Flutter) para automatizar procesos, mejorar la comunicación con los clientes y optimizar la gestión de envíos.

Planteamiento del Problema

A nivel regional, operan grandes empresas de transporte en toda la costa, ofreciendo precios y tiempos de entrega competitivos. Además, aplicaciones móviles conectan directamente a usuarios con conductores, ofreciendo soluciones instantáneas.

En Domitransp SAS, identificamos desafíos derivados de la falta de un sistema de información eficiente para gestionar procesos logísticos y administrativos. Algunos de estos problemas incluyen:

  • Recepción manual de solicitudes de envío a través de llamadas.
  • Problemas de escalabilidad al recibir un gran volumen de solicitudes u órdenes.
  • Comunicación ineficiente entre la empresa y el cliente debido a la saturación de llamadas.
  • Falta de información en tiempo real del estado de las solicitudes de los clientes.
  • Dificultad en el proceso mensual de facturación de la empresa.

El proyecto abordará estos problemas mediante la implementación de un software de gestión integral que optimizará todos los procesos que la empresa utilice.

Objetivo General

Desarrollar e implementar un software de gestión integral para Domitransp SAS que permita la administración eficiente de sus procesos logísticos y administrativos.

Objetivos Específicos

  • Aplicar métodos de recolección de datos en campo al stakeholder de la empresa.
  • Identificar problemas de desajuste en la gestión de envíos y administración de finanzas de los clientes de la empresa.
  • Diseñar la interfaz de la aplicación web y móvil.
  • Utilizar un lenguaje unificado de modelado para crear diagramas que respalden el desarrollo de la aplicación.
  • Implementar un ORM para la base de datos del aplicativo.
  • Desarrollar un módulo de envíos.
  • Crear un módulo de gestión de finanzas para el usuario.
  • Realizar pruebas en todas las secciones de la aplicación para identificar errores y excepciones que requieran corrección.
  • Implementar el aplicativo en sitio.

Implementación

El software se desarrollará utilizando Ruby on Rails para la aplicación web y Flutter para la aplicación móvil. Incluirá módulos para la gestión de envíos, administración de finanzas y otras funcionalidades clave.

Además, se implementaran servicios de AWS, como S3 y SES.
